# RSPort Frontend

This folder contains the RSPort storefront pages, static assets, and browser-side integration script.

The original static sports-shop pages have been upgraded with:

- API-backed products, cart, checkout, login, and account flows
- redesigned hero, navigation, cards, blog, footer, and login surfaces
- working blog article links through `blog-details.html`
- category links that route into filtered shop pages
- responsive visual polish through `assets/css/storefront.css`

Run the full project from the repository root:

```bash
npm start
```

Then open `http://localhost:3000`.
